개선된 3-Agent 구조

Agent 1: 검색 Agent
├─ 기본 검색 (템플릿 0~9번 항목)
│  - "{여행지} 항공권"
│  - "{여행지} 숙박"
│  - "{여행지} 통신"
│  └─ ...
│
└─ 특수사항 검색
   - "{여행지} 주의사항"
   - "{여행지} 여행 금지사항"
   - "{여행지} 필수 규정"
   - "{여행지} 입국 요건"

Agent 2: 템플릿 커스터마이징 Agent
- Agent 1의 "특수사항 검색" 결과 분석
- 중요한 특수 규정 발견 시 템플릿에 섹션 추가
- 예: "거주지 등록", "특정 지역 출입 금지", "사진 촬영 금지 구역" 등

Agent 3: 보고서 작성 Agent
- Agent 2가 만든 커스터마이징 템플릿 사용
- Agent 1의 검색 결과로 각 섹션 내용 작성
